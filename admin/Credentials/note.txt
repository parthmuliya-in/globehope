what can do for verification mail in anther gmail id
If you want to send OTP verification mail from another Gmail ID, you have 3 proper options.

✅ OPTION 1 — Use Another Gmail with App Password (Recommended)

If you want to use another Gmail like:

exampleadmin@gmail.com
🔹 Step 1: Enable 2-Step Verification

Go to:

👉 Google Account
Security → Turn ON 2-Step Verification

🔹 Step 2: Generate App Password

In Google Account:

Security → App passwords
Select:

App → Mail

Device → Other

Generate password

You will get 16-digit password like:

abcd efgh ijkl mnop

Remove spaces and use in PHPMailer.

🔹 Step 3: Update Your Code

Change this in send_otp.php:

$mail->Username = 'exampleadmin@gmail.com';
$mail->Password = 'abcdefghijklmnop'; // no spaces
$mail->setFrom('exampleadmin@gmail.com', 'Admin Login');

Done ✅

✅ OPTION 2 — Use Different Gmail Only for Receiving (No Code Change)

If you just want OTP to go to another Gmail ID:

Then simply:

Change admin email in your admins table

Example:

UPDATE admins SET email='newemail@gmail.com' WHERE id=2;

No SMTP change needed.

✅ OPTION 3 — Use Professional Email (Better for Production)

Instead of Gmail, use:

Domain email like: admin@yourdomain.com

SMTP from hosting (cPanel / Plesk)

Since you're using Plesk, this is better for real project.

In Plesk:

Go to Mail

Create email account

Use that SMTP details

Example:

$mail->Host = 'mail.yourdomain.com';
$mail->Username = 'admin@yourdomain.com';
$mail->Password = 'your_email_password';
$mail->Port = 587;
$mail->SMTPSecure = 'tls';
